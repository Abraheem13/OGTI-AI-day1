# Module 02 — ChatGPT vs Claude for Oil & Gas

**Why this module matters:** ChatGPT and Claude are both excellent — but they have **different personalities, different sweet spots, and different limits**. Choosing the wrong tool for the job wastes time and produces inferior output. This module teaches you to feel the difference in your hands.

## 🧠 The mental model — when to use which

| Job | Best tool | Why |
|-----|-----------|-----|
| Quick draft, brainstorm, code snippet | **ChatGPT** | Fast, fluent, breadth of training |
| Board memo, formal letter, regulator filing | **Claude** | Nuanced register, longer attention |
| Reading a 100-page document in one pass | **Claude** | Larger context window on free tier |
| Asking for the latest news | **Neither — use Perplexity** | Both have training cutoffs |
| Asking *only* about your uploaded documents | **NotebookLM** | Cites the exact page; no hallucination |

**Rule of thumb:** ChatGPT for *speed and drafting*. Claude for *nuance and long-document analysis*. Always compare outputs for important decisions.

## 🎓 Learning outcomes

By the end of this module you will:

1. Run the **same prompt** in both tools and articulate which output is better, and why
2. Choose the right tool for SOPs, long-document review, and incident investigation
3. Identify the *signature behaviours* of each tool (Claude's caution, ChatGPT's confidence)
4. Build a personal "which tool when" cheat sheet for your own role
